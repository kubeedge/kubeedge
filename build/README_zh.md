## tools
`certgen.sh` 用于生成证书和密钥的脚本,主要用于两个地方
- 生成cloud和edge的证书,生成的路径在 `/etc/kubeedge/ca`和`/etc/kubeedge/certs`
  ```
  certgen.sh genCertAndKey edge
  ```
- 用于生成`06-secret.yaml`
